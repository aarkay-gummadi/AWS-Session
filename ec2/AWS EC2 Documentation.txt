Amazon EC2 
-------------
* EC2 is nothing but Amazon Elastic Cloud Computing 
* It will provides scalable computing capacity in the AWS cloud
* It eliminates you need to invest hardware up front, so you can develop and deploy applications faster
* By using ec2 to launch as many or as few virtual servers as you need, configure security and networking and manage storage
* Reducing your need to forecast the traffic

**********************************
Features fo Amazon EC2 Instances
**********************************
```
Instance
----------
* Elastic computing environment is known as instance

AMI's 
-----
* AMI's is nothing but Amazon Machine Images
* Preconfigured Templates for your instance is known as AMI'scalable
* That package the bits you need for your server(including operating system and additional softwares)

Instance Type
--------------
* Various configurations of CPU, storage, memory and networking capacity for your instance is known as Instance Type

Key Pairs
----------
* Secure login information for your instance using Key Pairs(AWS stores the public key and you store the private key in a secure place)

Instance Storage Volumes
--------------------------
* Storage Volumes for temporary data that's deleted when you stop, hibernate or terminate your instance is known as instance storage volumes

Amazon EBS volumes
--------------------
* Persistent storage volumes for your data using Amazon Elastic Block Store is known as Amazon EBS volumes

Regions and Availability Zones
---------------------------------
* Multiple physical locations for your resources, such as insatnce, amazom ebs volumes is known as Regions and Availability Zones

Security Groups
---------------
* A firewall that can enables you to specify the protocols, ports and source IP ranges that can reach your instance using security groups

Elastic IP Addresses
--------------------
* Static IPv4 addresses for dynamic cloud computing is known as Elastic IP Addresses

Tags
----
* Metadata is known as Tags, that you can create and assign to your Amazon EC2 resources

VPC's
------
Virtual networks you can create that are logically isolated from the rest of the AWS cloud, and that you can optionally conneted to your own network is known as virtual private clouds
```
